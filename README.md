# Trader Performance vs Market Sentiment

## Overview
This project analyzes how trader performance varies with market sentiment (Fear/Greed).

## Methodology
1) Cleaned and merged trading + sentiment data
2) Created classification categories (Fear, Greed, etc.)
3)Analyzed PnL trends using groupby
4) Segmented trades based on size

## Key Insights
1) Higher profits observed during Extreme Greed
2) Larger trades show different behavior compared to smaller trades

## Tools Used
1)Python
2) Pandas
3) Matplotlib

## How to Run
Open the notebook in Google Colab and run all cells.
